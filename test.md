## 1. 인증 및 로그인 (Authentication)

### 1.1 로그인 기능
- [ ] 올바른 자격 증명으로 로그인 가능
- [ ] 잘못된 자격 증명으로 로그인 시 적절한 에러 메시지 표시
- [ ] JWT 토큰이 정상적으로 발급되고 저장됨
- [ ] 로그인 후 대시보드로 리다이렉트

### 1.2 세션 관리
- [ ] 페이지 새로고침 후에도 로그인 상태 유지
- [ ] 토큰 만료 시 자동 로그아웃
- [ ] 로그아웃 기능 정상 작동
- [ ] 로그아웃 후 보호된 페이지 접근 시 로그인 페이지로 리다이렉트

### 1.3 권한 검증
- [ ] Admin 전용 메뉴가 Admin 계정에만 표시
- [ ] Supervisor 권한 기능이 Supervisor와 Admin에게만 접근 가능
- [ ] 일반 사용자가 권한 없는 페이지 접근 시 차단

## 테스트 진행 방법

1. **테스트 계정 준비**
   - Admin 계정
   - Supervisor 계정
   - 일반 사용자 계정 (복수)

2. **테스트 데이터 준비**
   - 테스트용 부서
   - 테스트용 휴가 신청
   - 테스트용 급여 데이터

3. **테스트 순서**
   - 인증 → 사용자 관리 → 부서 → 휴가 → 급여 → 보고서 순으로 진행
   - 각 기능별로 정상 케이스와 예외 케이스 모두 테스트

4. **이슈 기록**
   - 발견된 문제점 상세 기록
   - 재현 방법 기록
   - 스크린샷 첨부
   - 우선순위 설정

## 테스트 완료 기준

- 모든 체크리스트 항목 통과
- Critical 이슈 0건
- Major 이슈 해결 또는 워크어라운드 확보
- 성능 기준 충족
- 보안 취약점 없음
